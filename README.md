# Shyam Saran P - Personal Portfolio

A modern, visually stunning personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Dark Theme**: Sleek black/charcoal background with neon blue/purple gradients
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop
- **Smooth Animations**: Framer Motion powered transitions and scroll animations
- **Glassmorphism**: Modern glassmorphism cards with blur effects
- **Interactive Elements**: Hover effects, typing animation, and micro-interactions
- **Single Page Application**: Smooth scrolling navigation between sections

## Tech Stack

- **Frontend**: React 18 with functional components
- **Styling**: Tailwind CSS for utility-first styling
- **Animations**: Framer Motion for smooth transitions
- **Icons**: Lucide React for consistent iconography
- **Build Tool**: Vite for fast development and building

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Preview production build**:
   ```bash
   npm run preview
   ```

## Sections

- **Hero**: Animated typing effect with role rotation, CTA buttons
- **About**: Personal bio and education details
- **Skills**: Categorized skills with icons
- **Experience**: Timeline-style work experience
- **Projects**: Card-based project showcase
- **Services**: Service offerings
- **Certifications**: Achievement highlights
- **Contact**: Contact form and social links

## Customization

- Update personal information in respective components
- Modify color scheme in Tailwind classes
- Add/remove sections as needed
- Replace placeholder resume.pdf with actual resume
- Update project links and descriptions

## Deployment

Ready for deployment on Vercel, Netlify, or any static hosting service.

```bash
npm run build
```

Upload the `dist` folder to your hosting provider.
